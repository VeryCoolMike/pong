How to install

1. RUST

Linux and MacOS
```
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Windows
Go to https://www.rust-lang.org/tools/install and follow the instructions

2. SDL2

Linux Fedora
```
sudo dnf install sdl2
```
